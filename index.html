Que 4 : Write a code to calculate area of a rectangle using inner function. In this case outer function should accept parameter length and inner function should accept parameter breadth.

Ans:

function outerfunction(length){
        function innerfunction(breadth){
          return length*breadth;
        }
        return innerfunction;
      }
      const length=3;
      const breadth=6;
      let area=outerfunction(length)(breadth);
      console.log(area);

Que5 : Take a variable in outer function and create an inner function to increase the counter every time it is called

Ans:

function outerfunction(x){
        function innerfunction(counter){
          return counter=counter+1;
        }
        return innerfunction;
      }
      let x=6;
      let counter=0;
      let count=outerfunction(x)(counter);
      console.log(count);

Que 7 : var a = 12;
        var x = (function () {
          var a = 12;
          return function () {
            alert(a);
          };
        })();
        x();

Ans:

        12
        Explanation : The variable x is assigned an immediately invoked function expression (IIFE) and inside the IIFE there is 
        another anonymous function which logs the value of variable a.When the IIFE function is called the current value
         of the anonymous function that is 12 as var a = 12 gets stored in the variable x and therefore the output is 12.

        (function outerFunc(outerArg) {

        outerArg = 123
